# Step 2 — Data class a instance (reseni)

## Co jsme udelali
- data class = strukturovany datovy model (jako YAML schema)
- instance = konkretni objekt podle sablony
- data class automaticky dava: toString(), equals(), copy()
- copy() vytvori kopii s upravenymi fieldy — original se nezmeni
